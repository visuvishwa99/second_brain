Cloze
{LangExtract} is an open-source library from Google for extracting structured data from unstructured text using LLMs.
END

START
Cloze
LangExtract provides {source grounding} (or provenance) by linking every extracted field back to its exact character span in the original text.
END

START
Cloze
To maintain consistency, LangExtract uses {schema-controlled output}, where you define the output structure via instructions and examples.
END

START
Cloze
LangExtract supports multiple model backends, including {Gemini}, OpenAI, and local models via {Ollama}.
END
