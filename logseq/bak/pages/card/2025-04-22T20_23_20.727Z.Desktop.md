- ## LEAD and LAG Functions
- LEAD:: Returns value from a subsequent row within the partition #card
- LAG:: Returns value from a previous row within the partition #card
- LEAD Syntax ->> `LEAD(column, offset, default) OVER (ORDER BY column)` #card
- ```
  SELECT month,
  sales,
  LEAD(sales, 1, "NA") OVER(
    ORDER BY month

- ## Recursive CTEs
- Recursive CTE Syntax ->> Has an anchor member and a recursive member combined with UNION ALL #card
- ```
  WITH RECURSIVE cte_name AS (
    -- Anchor member: initial query that returns base result set
    SELECT initial_columns FROM initial_table WHERE initial_conditions
    
    UNION ALL
    
    -- Recursive member: references the CTE itself
    SELECT recursive_columns FROM some_table 
    JOIN cte_name ON join_condition
    WHERE recursive_conditions
  )
  SELECT * FROM cte_name;
  ```
- Example:
	- ```
	  WITH RECURSIVE seq AS(SELECT 1 as n UNION ALLSELECT n + 1 FROM seq WHERE n < 5)SELECT * FROM seq
	  ```
